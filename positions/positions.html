ABSOLUTE

  1. will be moved w.r.t the parent container having position property set to some value which is not "static"
  2. top/bottom and left/right properties can be used
  3. z-index properties can also be used

STATIC(default)

  1. Being the default property value:
  2. top/bottom and left/right properties CANNOT be used
  3. z-index properties CANNOT be used

RELATIVE

  1. will be moved w.r.t its initial position on the page and will leave a void that cannot be filled by any peripheral elements
  2. top/bottom and left/right properties CAN be used
  3. z-index properties CAN be used

FIXED

  1. will be moved w.r.t the ><html> leaving a void that WILL BE FILLED BY PERIPHERAL ELEMENTS
  2. top/bottom and left/right properties CAN be used
  3. z-index properties CAN be used

STICKY [not a child of the container]

  1. setting position property value to "sticky" will not show any results.
  2. Will start moving only "top" property is being set
  3. top/bottom and left/right properties CAN be used
  4. z-index properties CAN be used
